🧘 Mindful Navigation - Extension Tests

Use this page to verify the extension is working correctly

📋 Before Testing:
  • Extension is loaded in Firefox
  • You can see the extension icon in toolbar
  • Press F12 to open Developer Tools
  • Keep Console tab open to see test results

Test 1: Extension Detection

Check if the Mindful Navigation extension is installed and active.

Test 2: Storage Access

Verify extension can read/write to storage.

Test 3: Background Script Communication

Test message passing between page and background script.

Test 4: Add Test Domain

Add "example.com" to your blocked domains list.

Test 5: Manual Navigation Test

Navigate to a test domain to trigger the overlay.

ℹ️ Make sure example.com is in your blocked list first (use Test 4)

🔍 Debugging Tips:
  • Check browser console (F12) for detailed logs
  • Go to about:debugging to inspect background script
  • Right-click extension icon → Inspect to debug popup
  • See TROUBLESHOOTING.md for common issues
